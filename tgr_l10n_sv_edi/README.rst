Facturación electrónica El Salvador (DTE)
==========================================

**Autor (metadata del módulo):** Julio Amaya

Módulo ``tgr_l10n_sv_edi``: integración con el flujo ``account_edi`` de Odoo para
generación, firma y envío de DTE al Ministerio de Hacienda de El Salvador.

Tipos soportados incluyen factura (01), crédito fiscal (03), notas, FSE (14) y
comprobante de donación electrónico CDE (15), según la configuración del diario
y el tipo de documento LATAM.
